ConvertAll Downloads

Online

ConvertAll can be run online without an installation. See convertall-web.bellz.org to try it out.

Android

For an Android device that conforms to the requirements page:

  • ConvertAll can be downloaded and installed from Google Play, here.
  • Alternately, it can be installed from an APK file: convertall_1.0.0.apk (22 MB)

Linux

On any 64-bit Linux OS, ConvertAll can be installed using the script included in:

  • convertall_1.0.0.tar.gz (3 MB)

Windows

On a Windows system that conforms to the requirements page, all that is needed is:

  • convertall_1.0.0.zip (12 MB)

Alternately, ConvertAll can be compiled from source — see the Installation page for more information.

Older Versions

Previous versions of ConvertAll are available from bellz.org/ConvertAll-py.
